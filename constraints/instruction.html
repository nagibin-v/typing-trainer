The file should be formatted as
<Title>
<Goal speed> <Goal accuracy>
<Text>
Goal speed should be an integer number from 0 to 200
Goal accuracy should be a float number from 0 to 1
Supported symbols are uppercase and lowercase English letters, digits, space, tab, newline symbol, and standard special symbols:
!@#&%^&*()-_=+[]{}/|;:'",<.>/?
